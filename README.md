# Employee Management System

## 📌 Project Overview
This is a Java-based Employee Management System developed using Core Java, JDBC, and MySQL.

## 🚀 Features
- Add Employee
- View Employee
- Update Employee
- Delete Employee

## 🛠 Technologies Used
- Core Java
- JDBC
- MySQL
- Eclipse IDE
- Git & GitHub
