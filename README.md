# Employee Management System

## 📌 Project Overview
The Employee Management System is a Java-based application developed using Core Java, JDBC, SQL, and MySQL. It is designed to manage employee records efficiently by performing CRUD (Create, Read, Update, Delete) operations.

## 🚀 Features
- Add Employee
- View Employee Details
- Update Employee Information
- Delete Employee Records
- Search Employee by ID
- Input Validations
- Exception Handling

## 🛠️ Technologies Used
- Core Java
- JDBC
- SQL
- MySQL
- Eclipse IDE

## ⚙️ How to Run the Project
1. Clone the repository.
2. Open the project in Eclipse IDE.
3. Create the MySQL database.
4. Update database username and password in the DBConnection class.
5. Add the MySQL JDBC Connector.
6. Run the Main class.

- Object-Oriented Programming (OOP)
- JDBC Connectivity
- CRUD Operations
- Exception Handling
- Input Validation
